fetchData()

Using fetch(url) and get companies data, then convert response to json. Next, Promise.all make sure that another data table is loaded and converted to json before next step which is modifying companies array. Now the algorithm for every company array row add some extra properties to object, which are calculated by functions declarated before Promise.all, getTotalIncome() and getLastIncome(). Function getData is used on line 20 with then(), where global variable is filled with complete data about companies and their incomes.
